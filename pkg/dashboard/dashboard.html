Schritt 2:

🔄 Neustart Schnell · ~5 Sek.

Stoppt FluxBot und startet ihn sofort neu. Nutze das wenn du Einstellungen im Dashboard gespeichert hast und sichergehen willst, dass alles übernommen wurde. Keine Daten gehen verloren.

🔨 Neu bauen Update · ~2 Min.

Lädt den neuesten Code herunter und kompiliert FluxBot komplett neu. Notwendig nach Updates vom Entwickler (neue Funktionen, Bugfixes). Dauert etwas länger als ein einfacher Neustart.

🌐 Browser-Cache leeren nach Rebuild: Nach einem Neu-Build zeigt der Browser manchmal noch das alte Dashboard. Drücke Strg + Shift + R (Windows/Linux) oder Cmd + Shift + R (Mac) – das lädt alles frisch.
