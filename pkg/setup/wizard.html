Willkommen bei FluxBot! 👋

Dein persönlicher KI-Assistent für Telegram, Discord und mehr. Die Einrichtung dauert ca. 5 Minuten.

  • 🔑 OpenRouter API Key Kostenlos unter openrouter.ai
  • 💬 Mindestens einen Messenger (Telegram, Discord, Slack oder Matrix)
  • ⏱️ Ca. 5 Minuten Zeit

🔑 KI-Provider & API Key

Wähle deinen KI-Anbieter aus und gib deinen API Key ein. Du kannst jeden Anbieter nutzen – oder bei mehreren Anbietern registriert sein.

Zugang zu Claude, GPT-4, Gemini, Perplexity und mehr – alles mit einem Key. → Kostenlos unter openrouter.ai
Die vollständige URL zum Chat-Completions-Endpunkt
Die genaue Modell-ID die dein Anbieter erwartet
Bitte wähle einen Anbieter und gib deinen API Key ein.
